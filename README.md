# Traductor Básico Inglés - Español en C#

 Descripción
Este proyecto consiste en el desarrollo de una aplicación en C# que funciona como un traductor básico entre los idiomas inglés y español.

El programa utiliza la estructura Dictionary para almacenar palabras y sus equivalencias.

# Funcionalidades

- Traducción de frases completas.
- Traducción parcial (solo palabras registradas).
- Opción para agregar nuevas palabras al diccionario.
- Menú interactivo en consola.

# Tecnologías utilizadas

- Lenguaje: C#
- Plataforma: .NET
- Editor: Visual Studio Code

# Ejecución del programa

Para ejecutar el programa:

dotnet run
